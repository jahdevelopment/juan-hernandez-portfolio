    Hernandez, J. (2022, March 11)

    Would you like to know what exactly HTML and CSS is?

    HTML

    "Hypertext Text Markup Language", commonly known as HTML, is the language created to represent the structure of websites; in other words, this is the skeleton of the apps. All the elements that compound a web page are implied in the content of this language. In each one of them, there is a way to represent the characteristics that connect the structure with other technologies.

    Every person interested in learning web design should start with HTML, it is easy to understand due to its simplicity, and helps the user to organize an ordered sequence of the elements.

    HTML Tag

    Tag is the mechanism to represent the existence of any element of the website. This is represented by two double brackets like this:

    Into the angle brackets, there is the space to put the sign of each element represented by a letter, some letters, or a word. The first pair of angle brackets (without / sign) represents the opening of some content, and the second pair (with the “/”) shows us the finalization of the element.

    HTML Element

    An element is one characteristic content in the structure of the HTML, represented by the tags, its name of it and other details that represent its functionality.

    HTML Atribute

    The attribute is the detail of the functionality of each element; also the attribute has the facility of being represented in different ways like id, class, type, colour, etc. Through this, the element could be connected with other languages with the objective of dynamizing its characteristics.
